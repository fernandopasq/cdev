# 🐳 C Dev Container Environment

A portable C development environment using Docker and VSCode Dev Containers.

---

## 🚀 Purpose

This project provides a fully reproducible environment for C development with:

- GCC compiler
- GDB debugger
- Make build system
- Linux environment via Docker

It is designed for learning, academic projects, and standardized C development setups.
